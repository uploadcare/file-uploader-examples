# File Uploader custom tab example

[![Edit js-custom-tab](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/github/uploadcare/file-uploader-examples/tree/main/examples/js-custom-tab/)

**Note:** The example in CodeSandbox does not work by default. You need to set Unsplash access key in .env first. 

## Run this demo locally

```bash
# clone this repo and go to the cloned folder
$ cd examples/js-custom-tab

# set .env with an editor of your choice

$ npm install
# or `yarn install`, if you wish

$ npm run start
# or `yarn start`
```
